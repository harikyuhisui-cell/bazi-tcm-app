# CLAUDE.md — 八字体質診断アプリ 開発指示書

## プロジェクト概要

四柱推命（八字）と東洋医学（中医学）を組み合わせた体質診断Webアプリ。
生年月日時から命式を算出し、五行バランスに基づく体質傾向と養生アドバイスを提供する。

**監修**: 鍼灸師（東洋医学の用語・概念の正確性を担保）

## コーディング規約

- TypeScript strict モード必須。`any` は禁止。
- 関数・変数名は英語。コメントは日本語可。
- Zod スキーマでAPIの入出力を必ず型検証する。
- 副作用のある関数（API呼び出し、DB操作）は `async/await` を使う。
- テストは `tests/` に配置し、命式算出ロジックは必ずユニットテストを書く。
- コンポーネントは `src/components/` に配置し、ページからのロジックを分離する。

## 制約事項（必ず遵守）

1. **医療行為と誤解される表現を絶対に避ける**
   - NG: 「この体質の人は〇〇病になりやすい」「治療には〇〇が効果的」
   - OK: 「東洋医学の観点では〇〇の傾向があるとされています」「養生の参考として」

2. **断定的な診断表現を避ける**
   - NG: 「あなたは水型体質です」（断定）
   - OK: 「水の気が強い傾向が見られます」（傾向・参考として）

3. **免責事項を常に表示する**
   - 結果画面には必ず「本アプリは占術・東洋医学の考え方に基づく参考情報を提供するものであり、医療診断ではありません」を表示する。

4. **個人情報の扱い**
   - 生年月日は診断に使用するが、外部に送信する際は最小限の情報に留める。
   - Claude APIに送信するデータには個人を特定できる情報を含めない。

## 東洋医学・四柱推命 用語規約

- 五行: 木・火・土・金・水（英語: Wood, Fire, Earth, Metal, Water）
- 十干: 甲乙丙丁戊己庚辛壬癸
- 十二支: 子丑寅卯辰巳午未申酉戌亥
- 陰陽: 陽（Yang）・陰（Yin）
- 体質分類: 気虚・血虚・陰虚・陽虚・気滞・血瘀・痰湿・湿熱（中医学九体質に準拠）
- 計算基準: 節入り（節気）基準で月柱を決定する（旧暦ではなく節気基準）

## Claude API 使用ガイドライン

- モデル: `claude-sonnet-4-6`（バランス重視）
- システムプロンプトには必ず東洋医学監修者の観点を含める
- レスポンスには必ず「参考情報である」旨を含めるよう指示する
- トークン節約のため、命式データは構造化JSONで渡す
