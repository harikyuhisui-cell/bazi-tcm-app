## 1. 肝鬱気滞（かんうつきたい）

**概要**: 気の巡りが滞りやすく、ストレスの影響を受けやすいタイプ。

| 項目 | 内容 |
|---|---|
| 性格傾向 | 責任感が強く抱え込みやすい／完璧主義／イライラと憂うつの波／我慢後の感情爆発／対人ストレスに敏感 |
| 身体傾向 | 胸脇部の張り／ため息／梅核気／月経前の胸の張り・気分の波／慢性的な肩首のこり／ストレス性の食欲増減 |
| 食養生（推奨） | しそ・みつば・セロリ・陳皮／ジャスミン茶・ミント／春菊・菜の花／適度な酸味 |
| 食養生（控える） | 脂物の摂りすぎ／飲酒でのストレス発散／強い辛味の過剰／やけ食い・早食い |
| 生活（推奨） | 深呼吸・胸郭ストレッチ／律動的な運動／予定の余白／感情の言語化／香りでのリラックス |
| 生活（控える） | 怒りの溜め込み／休日も仕事を考え続ける／夜更かし／欠食 |
| 受診勧奨 | 抑うつ・不安が2週間以上／頻繁な胸痛・動悸／長期の食欲不振・不眠 → 心療内科・内科 |

**経穴**: LR3 太衝（肝経原穴・疏肝理気）／GB34 陽陵泉（筋会・脇肋部の張り）／PC6 内関（寛胸理気）／LI4 合谷（四関穴として太衝と組み合わせ）

## 2. 心火亢盛（しんかこうせい）

**概要**: 心の火が高ぶりやすく、興奮・ほてり・不眠が出やすいタイプ。

| 項目 | 内容 |
|---|---|
| 性格傾向 | 情熱的で行動力がある／高ぶると止まらない／せっかち／興奮による不眠／喜怒哀楽が明瞭 |
| 身体傾向 | 顔面紅潮・のぼせ／口内炎・口渇／入眠困難・多夢／動悸／舌尖紅／五心煩熱の傾向 |
| 食養生（推奨） | ゴーヤ・緑茶・たけのこ（苦味）／蓮の実・百合根／涼性の夏野菜／小豆・はと麦 |
| 食養生（控える） | 温熱性香辛料の過剰／蒸留酒の過飲／夕方以降のカフェイン／揚げ物・焼き肉の頻食 |
| 生活（推奨） | 就寝前のデジタルデトックス／夕方以降は穏やかな運動／短い昼休憩／瞑想・呼吸法 |
| 生活（控える） | 深夜の仕事・興奮する娯楽／徹夜／怒りに任せた言動／熱いサウナ・長風呂 |
| 受診勧奨 | 安静時の動悸・脈の乱れ／胸痛・圧迫感／1か月以上の不眠／高血圧の持続 → 循環器内科・内科 |

**経穴**: HT7 神門（心経原穴・安神）／PC8 労宮（滎火穴・清心熱）／KI1 湧泉（引火帰元）／PC6 内関（寧心安神）

## 3. 脾虚湿盛（ひきょしっせい）

**概要**: 消化吸収の力が弱く、余分な湿を溜め込みやすいタイプ。

| 項目 | 内容 |
|---|---|
| 性格傾向 | 思い悩みやすい／心配性で決断に時間がかかる／穏やか／集中時の欠食または間食増／雨天時の気分低下 |
| 身体傾向 | 食後の眠気・胃もたれ／軟便・下痢傾向／身体の重だるさ・下肢のむくみ／歯痕舌／甘味への欲求／湿気での悪化 |
| 食養生（推奨） | 山芋・かぼちゃ・米／はと麦・小豆・コーン茶／生姜・ねぎ／温かい加熱調理 |
| 食養生（控える） | 冷飲の常飲／生ものの摂りすぎ／甘いお菓子の頻食／乳製品・脂物の過剰 |
| 生活（推奨） | よく噛んで腹八分目／食後の軽い散歩／除湿と保温／思考の書き出し／温灸セルフケア |
| 生活（控える） | 欠食・不規則な食事／冷房下での冷飲継続／夜遅くの食事／悩みながらの食事 |
| 受診勧奨 | 体重減少を伴う食欲不振／2週間以上の下痢／血便・黒色便／急激なむくみ・乏尿 → 消化器内科・内科 |

**経穴**: ST36 足三里（健脾益気・養生灸の代表）／SP6 三陰交（健脾・水分代謝）／SP9 陰陵泉（利水滲湿）／CV12 中脘（胃募穴・調理脾胃）

## 4. 肺気虚（はいききょ）

**概要**: 呼吸と防衛の気が不足し、風邪をひきやすいタイプ。

| 項目 | 内容 |
|---|---|
| 性格傾向 | 繊細で感受性豊か／悲しみ・寂しさを感じやすい／控えめ／気候変化に敏感／丁寧で几帳面 |
| 身体傾向 | 易感冒・治りにくい／声が小さい・話し疲れ／自汗／アレルギー性鼻炎／皮膚の乾燥／息切れしやすい |
| 食養生（推奨） | 梨・白きくらげ・れんこん（白い食材）／はちみつ・山芋／銀杏・くるみ／生姜・ねぎ・しそ |
| 食養生（控える） | 冷たく乾いた食品／強い辛味の過剰（耗気傷陰）／タバコ／冷飲の一気飲み |
| 生活（推奨） | 腹式呼吸の習慣／軽い有酸素運動／加湿とマスク／首・背中上部の保温／十分な睡眠 |

**経穴**: LU9 太淵（肺経原穴・補肺気）／BL13 肺兪（背部兪穴）／ST36 足三里（培土生金）／LI4 合谷（表裏経・衛気を助ける）

## 5. 腎陰虚（じんいんきょ）

|---|---|

| 身体傾向 | 五心煩熱／盗汗（寝汗）／夜間の口渇／腰膝のだるさ／耳鳴り・めまい／皮膚・毛髪の乾燥 |
| 食養生（推奨） | 黒ごま・黒豆・黒きくらげ／牡蠣・帆立など滋陰の魚介／梨・百合根・白きくらげ／クコの実・桑の実 |
| 食養生（控える） | 温燥性香辛料の過剰／飲酒過多／カフェイン過剰／塩分過多 |

| 生活（控える） | 徹夜・夜更かしの常態化／過度なサウナ発汗／激しい運動での消耗／長時間労働の常態化 |

## 7. 気血両虚（きけつりょうきょ）

**概要**: エネルギーと栄養の両方が不足しやすいタイプ。

| 項目 | 内容 |
|---|---|
| 性格傾向 | 気持ちに体力が追いつかない／疲労時の不安感／物忘れ・集中力低下／穏やかだが無理がきかない／人混みで消耗 |
| 身体傾向 | 顔色が青白い／めまい・立ちくらみ／易疲労・回復が遅い／動悸・息切れ／爪・髪のトラブル／月経量減少／浅い眠り・多夢 |
| 食養生（推奨） | なつめ・クコの実・竜眼肉／鶏肉・卵・レバー／ほうれん草・にんじん・黒ごま／米・山芋・かぼちゃ／煮込みスープ・薬膳粥 |
| 食養生（控える） | 極端な食事制限／冷たいもの・生ものの過剰／カフェインでのごまかし／欠食しての長時間労働 |
| 生活（推奨） | 睡眠最優先（7時間以上）／散歩から始める運動／疲れ切る前の休憩／目の酷使を控える／規則正しい温食 |
| 生活（控える） | 徹夜作業／激しい運動での消耗／長時間の画面注視／気合いで乗り切る無理 |
| 受診勧奨 | 頻繁な立ちくらみ・めまい／蒼白＋動悸・息切れ（貧血疑い）／月経異常の持続／1か月以上の強い倦怠感 → 内科・婦人科 |

**経穴**: ST36 足三里（補気の代表）／SP6 三陰交（養血）／CV6 気海（補益元気）／BL20 脾兪（気血生化の源を助ける）

## 8. 痰湿瘀血（たんしつおけつ）

**概要**: 余分な水分・脂質と血の滞りが重なりやすいタイプ。

| 項目 | 内容 |
|---|---|
| 性格傾向 | おおらかでマイペース／習慣を変えるのが苦手／腰が重いが持続力あり／食でのストレス発散／身体のサインに気づきにくい |
| 身体傾向 | 頭重・身体の重だるさ／慢性的な肩首こり／肌のくすみ・クマ・シミ／固定性の刺痛傾向／舌質暗紫・舌下静脈怒張／月経痛・血塊／低気圧での悪化 |
| 食養生（推奨） | 玉ねぎ・らっきょう・黒きくらげ／青魚／はと麦・冬瓜・海藻／酢・山椒・ターメリック／紅花茶・よもぎ茶 |
| 食養生（控える） | 脂物・揚げ物・スナックの頻食／甘味・加糖飲料の常飲／冷たいものの過剰／飲酒過多／夜遅くの重い食事 |
| 生活（推奨） | 毎日20〜30分の有酸素運動／入浴で温め巡らせる／1時間ごとに動く／体重・腹囲の記録／禁煙 |
| 生活（控える） | 座りっぱなし／冷えの放置／痛み・こりの放置／夜食・間食の習慣化 |
| 受診勧奨 | 突然の胸痛・激しい頭痛（救急）／片側のしびれ・脱力・構音障害（救急）／悪化し続ける固定痛／しこりの自覚／健診での血圧・血糖・脂質異常 → 内科・循環器内科 |

**経穴**: SP10 血海（活血）／ST40 豊隆（化痰の代表穴）／LR3 太衝（理気活血）／SP6 三陰交（血・水両面）

## 補足（実装メモ）

- 経穴コードは WHO standard acupuncture point locations（2008）の略号に準拠しています。
- 全タイプ共通で「診断」という語を避け、「タイプ」「傾向」と表現しています。
- warningSignals は「セルフケアの範囲を超えるサイン」として受診勧奨を明示しています。
- 修正のご指示は本ファイルへの直接コメントでも、口頭でも構いません。反映後に再レビューをお願いします。
